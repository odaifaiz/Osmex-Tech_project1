# 📱 City Fix

**City Fix** is a mobile application designed to empower the local community of Taiz to report poor public services directly to the responsible authorities using real-time photos.

The app aims to accelerate the process of identifying issues and improving city services through citizen participation.

---

## ✨ Features

- 📸 Report issues using live photos
- 📍 Attach location to reports
- 🏙️ Focused on improving public services in Taiz
- 🤝 Community-driven problem reporting
- ⚡ Faster communication with authorities

---

## 🎯 Purpose

City Fix connects citizens with local authorities to:

- Report infrastructure problems
- Highlight service failures
- Encourage transparency
- Improve response times
